# Basic-Bug-Bounty-Tools
This Script install the basic Bug bounty tools in one go 
- For Smooth Installation Make sure you have Go-lang Installed in machine.
- Make sure you are Running this script in Root.

   - Subfinder
   - Httpx
   - Nuclei
   - Naabu
   - Katana
   - DNSX
   - Notify
   - Fuzzing Templates
   - Uncover

2. Tomnomnom Tools

3. Arjun
